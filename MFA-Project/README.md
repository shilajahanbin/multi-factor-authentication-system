# Multi-Factor Authentication System

## Overview

This project implements a Multi-Factor Authentication (MFA) system using:

- Username and Password Authentication
- Email OTP Verification
- Duo Security Push Authentication

The system demonstrates modern authentication workflows and improves account security through multiple verification layers.

## Technologies Used

### Backend
- Python
- Flask
- MongoDB
- JWT Authentication
- Duo Security API

### Frontend
- React Native
- Expo

## Features

- User Registration
- User Login
- Email OTP Verification
- Duo Push Authentication
- JWT Session Management
- Secure Authentication Flow

## System Architecture

Frontend (React Native)
↓
Flask API Backend
↓
MongoDB Database
↓
Duo Security Authentication
